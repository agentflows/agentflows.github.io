ArXiv Paper Searcher Tool

A tool that searches arXiv for papers based on a given query.

Tool Metadata

Input Types:

  • query: str - The search query for arXiv papers.
  • size: int - The number of results per page (25, 50, 100, or 200). If None, use 25.
  • max_results: int - The maximum number of papers to return (default: 25). Should be less than or equal to 100.

Output Type:

list - A list of dictionaries containing paper information.

Demo Commands:

execution = tool.execute(query="tool agents with large language models")

Search for papers about tool agents with large language models.

execution = tool.execute(query="quantum computing", size=100, max_results=50)

Search for quantum computing papers, with 100 results per page, returning a maximum of 50 papers.

execution = tool.execute(query="machine learning", max_results=75)

Search for machine learning papers, returning a maximum of 75 papers.

User Metadata:

  • Valid sizes: [25, 50, 100, 200]
  • Base URL: https://arxiv.org/search/
